Dovednosti

V HTML umím tvořit statické webové stránky. Ovládám řádkové i blokové elementy, nicméně v dnešní době, kdy je nutné, aby obsah webu byl plně responzivní, HTML vnímám jako poměrně zastaralý způsob tvorby webových stránek.

V CSS (kaskádových stylech) můžeme plně ovlivnit design webové stránky aneb to, jak stránka vypadá. CSS ovládám včetně zápisů pravidel pro plně responzivní web.

C# je můj hlavní programovací jazyk. Ovládám tvorbu aplikací v konzoli a nyní se učím i tvorbu webových aplikací s využitím ASP.NET Core MVC.


JavaScript nám přidává do stránek dynamiku. Kdykoliv chcete rozbalit nějaký dropdown nebo si přejete zobrazit vyskakovací okno, je to práce JavaScriptu. JavaScript zkrátka Vaše stránky rozpohybuje.

S databází komunikujeme jazykem SQL, který nám umožňuje tvořit databázové dotazy. Ovládám napojení na databázi jak v konzolových aplikacích, tak v těch webových.

Bootstrap je framework od Twitteru, který nám značně zjednodušuje tvorbu webových stránek pomocí zápisu jednotlivých tříd. Je plně responzivní a rád jej používám.
